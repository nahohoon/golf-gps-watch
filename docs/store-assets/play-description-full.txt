K2 Golf GPS — Wear OS 골프 거리 앱

K2 Golf GPS는 스마트워치에서 현재 위치와 그린 중심 좌표를 비교해, 그린까지 남은 거리를 meter(m) 단위로 보여주는 Wear OS 전용 앱입니다.

■ 주요 기능
• 그린까지 거리(m) 표시
• 18홀 홀 번호 표시 및 이전/다음 홀 이동
• 스와이프로 빠른 홀 변경
• 새로고침으로 GPS 위치 재조회
• GPS 수신 상태 및 정확도(±m) 표시
• 마지막 갱신 시각 표시

■ Wear OS에 맞춘 사용성
• 워치 단독 실행(standalone) — 폰 없이도 사용 가능
• 검정 OLED 배경, 큰 거리 숫자 중심 UI
• 터치 버튼과 스와이프 제스처 지원

■ 개인정보·연결
• 위치 권한은 거리 계산에만 사용합니다
• 사용자 위치를 서버로 전송하지 않습니다
• 위치 기록을 앱 서버에 저장하지 않습니다
• 인터넷 서버 통신 없이 기기 내에서 거리를 계산합니다
• 광고 SDK 없음, 계정 가입 없음

■ 이용 안내
• GPS 신호와 기기 센서 상태에 따라 표시 거리와 정확도는 달라질 수 있습니다
• 실제 라운드 전 워치에서 GPS 수신 상태를 확인해 주세요
• 코스 좌표는 앱에 포함된 데이터를 기준으로 합니다

K2 Golf GPS로 워치에서 간단하게 그린까지 거리를 확인해 보세요.
