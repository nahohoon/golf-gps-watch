K2 Golf GPS 개인정보처리방침

앱명: K2 Golf GPS · 패키지: com.golfgps.watch

시행일: 2026년 5월 28일

1. 개요

K2 Golf GPS(이하 “앱”)는 Wear OS 스마트워치에서 골프 라운드 중 그린까지의 거리를 meter(m) 단위로 표시하기 위한 앱입니다. 본 방침은 앱이 처리하는 정보와 그 이용 목적을 설명합니다.

2. 수집·이용하는 정보

앱은 다음 정보를 기기 내에서만 이용합니다.

  • 위치 정보(GPS): 현재 위도·경도 — 선택한 홀의 그린 중심까지 직선 거리 계산에 사용
  • GPS 정확도: 수신 품질 표시(±m)에 사용

앱은 계정, 이메일, 전화번호 등 개인 식별 정보를 수집하지 않습니다.

3. 위치 권한 사용 목적

  • ACCESS_FINE_LOCATION / ACCESS_COARSE_LOCATION: 실시간 GPS 좌표 수신
  • 앱 실행·새로고침·홀 변경 시 거리 재계산
  • 백그라운드 지속 추적 없음 — 사용자 조작 시에만 위치 요청

4. 저장·전송·공유

  • 사용자 위치를 외부 서버로 전송하지 않습니다
  • 위치 기록을 앱 운영자 서버에 저장하지 않습니다
  • 제3자 광고·분석 SDK를 사용하지 않습니다
  • 인터넷 서버 통신 권한을 사용하지 않습니다(오프라인 거리 계산)

그린 좌표 등 코스 데이터는 앱에 포함된 데이터를 기준으로 하며, 거리 계산은 기기 내에서 수행됩니다.

5. 보관 기간

앱은 사용자 위치를 별도의 서버 DB에 보관하지 않습니다. 마지막 갱신 시각 등은 라운드 중 화면 표시 목적으로만 앱 실행 중 메모리에 유지됩니다.

6. 이용자 권리

위치 권한은 Android 설정에서 언제든 허용·거부할 수 있습니다. 권한을 거부하면 거리 표시 기능을 사용할 수 없습니다.

7. 아동

본 앱은 만 14세 미만 아동을 대상으로 하지 않으며, 아동 정보를 고의로 수집하지 않습니다.

8. 방침 변경

본 방침은 앱 또는 관련 법령 변경 시 수정될 수 있으며, 변경 시 본 페이지에 게시합니다.

9. 문의

개인정보 관련 문의는 Play Store 앱 페이지의 개발자 연락처를 이용해 주세요.
